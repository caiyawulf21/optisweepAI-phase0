# Phase 0 Scope

Phase 0 supports only CAT-1: WCS / Service Failure.

It validates:

- symptom-driven workflow selection
- grounded retrieval over curated CAT-1 records
- YAML-based role-aware troubleshooting
- deterministic escalation
- offline reusable procedure and workflow drafting

It does not include production deployment, enterprise connectors, automated ingestion, MCP, Entra ID auth, ML classifiers, knowledge graphs, or multi-category troubleshooting.
