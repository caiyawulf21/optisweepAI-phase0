# Extract CAT-1 Knowledge Record

Extract only evidence-supported CAT-1 WCS / Service Failure facts from the provided local source material.

Return structured fields for:

- observable symptoms
- issue category
- failure signature
- observed signals
- affected components
- RCA or suspected root cause
- resolution summary
- historical resolution steps
- escalation notes
- source evidence references

Do not invent operational steps. Detailed procedures belong in workflow YAML or procedure candidates supported by incident evidence.
